تطبيق ويب متطور لبناء السير الذاتية التفاعلية، يهدف المشروع إلى تبسيط عملية إنشاء ملفات تعريف مهنية جذابة بلمسة فنية احترافية وتقنيات برمجية حديثة[cite: 1].

---

## ✨ المميزات الرئيسية (Features)

* **⚡ المعاينة الحية (Live Preview):** تجربة تحرير سلسة تتيح لك رؤية التعديلات على القالب فور إدخال البيانات[cite: 1].
* **🎨 تصميم عصري (Dark & Gold Aesthetic):** واجهة مستخدم تعتمد السمة السوداء والذهبية التي تعكس الأناقة والاحترافية[cite: 1].
* **📄 تصدير عالي الجودة:** إمكانية تحميل السيرة الذاتية بصيغة PDF جاهزة للطباعة وبمقاس A4 قياسي[cite: 1].
* **⚙️ إدارة مرنة للمحتوى:** نظام متكامل لإضافة وترتيب أقسام الخبرة العملية، التعليم، والمهارات التقنية[cite: 1].
* **🌍 دعم RTL/LTR:** يدعم المشروع اللغتين العربية والإنجليزية بشكل كامل لتلبية احتياجات السوق المحلي والعالمي[cite: 1].

## 🛠 التقنيات المستخدمة (Tech Stack)

* **Framework:** [Next.js 14+](https://nextjs.org/) (App Router)[cite: 1].
* **Styling:** [Tailwind CSS](https://tailwindcss.com/) لتصميم واجهات سريعة ومتجاوبة[cite: 1].
* **State Management:** [Zustand](https://zustand-demo.pmnd.rs/) لإدارة حالة البيانات بكفاءة عالية[cite: 1].
* **PDF Engine:** استخدام `@react-pdf/renderer` لضمان دقة الملفات المصدرة[cite: 1].
* **Icons:** مكتبة [Lucide React](https://lucide.dev/) لتوفير أيقونات بسيطة ومعبرة[cite: 1].
